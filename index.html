Selected Work

Four projects that show how I approach UX, UI, and implementation.

UX Research & Strategy

Strategic Architecture

Before the UI, comes the logic. I built Afous Essence from the ground up, merging brand identity with a rigorous UX strategy. By mapping user intent through research and information architecture, I engineered a high-conversion journey that prioritizes a seamless path to purchase.
UX Research & UI Strategy

Balancing Aesthetics and Conversion

A critical analysis of the Zara Home e-commerce ecosystem. I identified friction points where the "magazine-style" UI compromised usability, providing architectural solutions that protect the brand's high-fashion DNA while optimizing the user journey for conversion.
Visual Design & Art Direction

High-Fashion Digital Presence

A conceptual restyling of a luxury retail platform. I challenged the boundaries of standard e-commerce layouts to create a high-impact visual identity, focusing on editorial typography and sophisticated spacing without sacrificing core functional logic.
Front-end Development

Responsive Architecture from Scratch

A multi-page implementation focused on clean, semantic code. I built a responsive travel interface using pure HTML and CSS to master layout fluidity and modern browser behaviors, ensuring a seamless experience across all screen sizes.
